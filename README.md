# BlazorNet10Demo (2026 Edition)

A clean, modular, .NET 8/9 Blazor Web App demonstrating modern architecture patterns using:

- Domain‑Driven Design (DDD) principles  
- Entity Framework Core  
- Repository/DataAccess layer  
- Blazor Server Interactivity  
- Syncfusion UI components (future integration)  

This project is structured for clarity, maintainability, and educational purposes — ideal for tutorials, demos, and hands‑on learning.

---

## 📁 Solution Structure

BlazorNet10DemoSolution
├── BlazorNet10Demo.Domain        # Domain models, enums, validation
├── BlazorNet10Demo.DataAccess    # EF Core DbContext, repositories
├── BlazorNet10Demo.UI            # Blazor Web App (Server Interactivity)
├── .github                       # GitHub workflows (CI/CD)
└── README.md                     # Project documentation


---

## 🎯 Project Goals

- Demonstrate a clean, layered architecture  
- Build a Task Management demo using DDD concepts  
- Use EF Core for persistence  
- Build a modern Blazor UI with Syncfusion components  
- Provide a step‑by‑step learning path for .NET developers

## 🚀 Getting Started

### 1. Clone the repository

git clone https://github.com/akoranteng/BlazorNet10Demo_2026_v2.git


### 2. Open the solution

Open `BlazorNet10DemoSolution.slnx` in Visual Studio 2022.

### 3. Run the application

Set **BlazorNet10Demo.UI** as the startup project and press **F5**.
